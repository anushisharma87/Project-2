# Customer Segmentation using PCA and K-Means

## Objective

Perform customer segmentation using Unsupervised Learning.

## Features

- Read customer dataset
- Standardize data
- Elbow Method
- Silhouette Score
- Principal Component Analysis (PCA)
- K-Means Clustering
- Cluster Visualization
- Customer Personas
- Save clustered data

## Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn

## Run

python main.py
